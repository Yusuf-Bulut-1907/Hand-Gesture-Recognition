# Hand Gesture Recognition

## Introduction

The objective of this project is to develop and evaluate an automated system for recognizing hand-drawn numerical sketches in a 3D space. Using spatial coordinate sequences $$r(t) = [x(t), y(t), z(t)]^T$$, the system identifies digits from 0 to 9 to enable seamless human-computer interaction.
Acting as Data Scientists, our mission is to implement, benchmark, and compare traditional signal processing baselines against state-of-the-art sequential models. A core focus of this study is the analysis of model robustness across two distinct scenarios: **User-Dependent** (personalized recognition) and **User-Independent** (generalized recognition).

## Getting Started

To begin working on the project, please follow the steps below to set up the development environment.

### Prerequisites

Ensure that the following tools are installed on your system:

* **Python 3.9 or higher** ([download](https://www.python.org/downloads/))
* **pip** (Python Package Manager) ([installation guide](https://pip.pypa.io/en/stable/installation/))
* **git** (To clone the repository) ([download](https://git-scm.com/install/))

### Step 1: Clone the Repository

Open your terminal or command prompt and clone the repository:

```bash
git clone https://github.com/Yusuf-Bulut-1907/Hand-Gesture-Recognition.git
cd Hand-Gesture-Recognition
```

### Step 2: Install Python Libraries

Install all necessary project libraries listed in the `requirements.txt` file:

```bash
pip install -r requirements.txt
```
